Incremental: Advantages and Disadvantages

Advantages Disadvantages
➕ The software will be generated quickly during the software life cycle. ➖ It requires a good planning designing.
➕ This model is less costly compared to others. ➖ Problems might cause due to system architecture as such not all requirements collected up front for the entire software lifecycle.
➕ Throughout the development stages changes can be done. ➖ Each iteration phase is rigid and does not overlap each other.
➕ It is flexible and less expensive to change requirements and scope. ➖ Rectifying a problem in one unit requires correction in all the units and consumes a lot of time.
